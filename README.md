# install

## examples - maybe you need to do it a little bit different
```sh
sudo apt-get install espeak 

sudo dpkg -i  ./autokey-common_0.96.0_all.deb 
sudo dpkg -i  ./autokey-qt_0.96.0_all.deb
```
copy and start
